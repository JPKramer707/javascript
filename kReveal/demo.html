kReveal Demo

Reveals hidden page elements on click (or other action)

You have any number of hidden elements (revealees) and any number of other elements which, when used, reveals the revealees.

Simply add a unique className to revealees, such as "revealMe." Then, add a classname to the revealers such as "kReveal show_revealMe" which essentially instructs kReveal to show all elements with a class of "revealMe".

You can also customize what type of reveal will be done - default is a simple display:block, but any method can be called via callback function.
