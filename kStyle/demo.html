kStyle Demo

kStyle provides an easy interface to alter CSS styles, rather than just the style given on an element. This lets you make page-wide changes with a single stroke. For example, all links in this page are flashing (or should be.) This is not accomplished by finding every link in the page, remembering it, and looping through them to set their colors in unison. Rather, this is done simply by hooking into the CSS selector that points to them, and changing one of its rules.

The theory is that this will speed up your script considerably, since more of the work is offloaded to native browser code.
